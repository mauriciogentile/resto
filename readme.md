To run the app:

1. Install node.js
2. cd to folder justeat/src
3. Run the following command:

	> npm install (it should install express web server)

4.To run the app run the command:

	> node app.js

5. Open browser on http:\\localhost:3000


To run the tests:

1. Install karma

	> npm install –g karma@0.8.7

2. Run test script:
	
	> scripts\test.bat
